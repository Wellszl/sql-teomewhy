# SQL - TeoMeWhy

Exercícios e anotações do curso de SQL do TeoMeWhy.

## Estrutura

```
aulas/         -> conteúdo visto em cada aula, organizado por tema
exercicios/    -> listas de exercícios resolvidas, organizadas por assunto
```

Os arquivos são nomeados seguindo a ordem em que os temas foram aprendidos, do mais básico ao mais avançado.

## Observação

Os arquivos `.csv` e o banco `.db` usados nos exercícios não estão neste repositório, pois contêm dados reais de pessoas (ignorados via `.gitignore`). Para rodar as queries localmente, é necessário ter esses arquivos por conta própria.
